# PART 1. 플랫폼 전체 구조

## 1. SPA199란

두바이 출장(온디맨드) 마사지 **리스팅 마켓플레이스**. 독립 프리랜서 테라피스트를 플랫폼에 등재하고, 고객이 온라인으로 검색→예약→결제하면 테라피스트가 호텔/자택/오피스로 방문해 서비스를 제공한다. 테라피스트는 직원이 아닌 **독립 사업자**이며, 플랫폼은 중개 수수료를 받는다.

## 2. 세 가지 사용자와 기능

### 2.1 고객 (Customer)
- **탐색**: 16개 시술 카테고리, 지역(13개)·가격·시술 필터, 리스트/지도 토글, 평점·리뷰·경력 표시
- **프로필**: 자기소개, 자격증, 갤러리(클릭 확대), 서비스별 메뉴 사진, 실고객 리뷰
- **예약**: 4단계 위저드 — 서비스 선택 → 가용 시간 캘린더(중복예약 자동차단) → 주소+방문메모 → 결제. 첫 예약 20% 자동 할인
- **예약 관리**: 내역/상세, 취소(48h+ 100% / 48–24h 50% / 24h내 0% 자동 환불계산), 완료 후 리뷰 작성, 문의/신고, 재예약 버튼
- **기타**: 인앱 알림(예약 상태 변화 자동), 4개 언어(영어/아랍어 RTL/한국어/태국어), 리뷰 신고

### 2.2 테라피스트 (Therapist)
- **프로필 관리**: 소개글, 경력, 자격증, 전문 시술, 서비스 지역 — **가격은 테라피스트 본인이 설정**
- **사진**: 프로필 1장 + 갤러리 10장 + 서비스별 메뉴 사진 — **전부 admin 검수 후 공개** (검수 대기/승인/반려 + 자동 알림)
- **스케줄**: 주간 가용시간 템플릿 + 휴무일, 예약 확정 시 슬롯 자동 차단
- **예약 처리**: 요청 수락/거절 (12시간 미응답 시 자동 만료·고객 무과금), 세션 종료 후 완료 처리
- **정산**: 완료 세션 기준 매출 80% (플랫폼 수수료 20%), 정산 내역 대시보드

### 2.3 관리자 (Admin)
- **KPI 대시보드**: 예약 수, GMV, 취소율, 환불액, 수수료 수익
- **테라피스트 승인 게이트**: 신규 가입자는 검토 후 승인해야 검색 노출 (Approve/Unlist)
- **사진 검수 큐**: 업로드 사진 미리보기 → 승인(즉시 공개)/반려(자동 통보)
- **리뷰 검수**: 게시/숨김, 신고된 리뷰 처리
- **사용자 관리**: 계정 정지/복구
- **문의/신고 처리**: 답변 + 해결 처리 (고객에게 표시)
- **공지/프로모션** 게시

## 3. Trust & Safety (구축 완료)

| 장치 | 내용 |
|------|------|
| 승인 게이트 | 무검증 테라피스트 노출 원천 차단 |
| 사진 검수 | 선정적/오프폴리시 이미지 사전 차단 — 가이드: 시술/작업공간/프로페셔널 사진만 |
| 리뷰 신고 + 검수 | 부적절 리뷰 신고→admin 처리 |
| 예약 단위 신고 | 노쇼/품질/부적절 행위 신고 채널 (양방향) |
| 약관 | "엄격히 치료 목적 마사지만, 부적절 행위 시 즉시 종료+영구 퇴출" 명문화 |
| 법적 페이지 | Terms / Privacy / Refund Policy 게시 (Stripe 심사 요건 충족) |

## 4. 결제 — 현재와 계획

- **현재**: 데모 결제 (실과금 없음, 플로우 검증용)
- **계획 (Stripe)**: 예약 시 카드 **홀드만** → 테라피스트 수락 순간 실청구 → 거절/만료 시 홀드 해제(고객 무과금). 부분환불 API로 48/24h 정책 자동 집행. 카드+Apple Pay+Google Pay, AED
- **정산 구조**: 파트너 법인이 결제 수취(merchant of record) → 테라피스트에 80% 주간 정산. 분배율은 코드에 고정되어 투명
- 규모 확대 시 Stripe Connect(테라피스트별 자동 분배)로 전환 가능

## 5. 인프라·운영

- 스택: Next.js + Supabase(DB/인증/스토리지) + Vercel 호스팅 — 유지비 월 ~$45 수준
- GA4 분석 연결됨, SEO(sitemap + 지역별 랜딩 13페이지), 예약 자동만료 크론
- GitHub 자동 배포, 다국어 4개

# PART 2. 파트너 판단 필요 사항 (Decision Items)

> 아래 9개는 기술이 아닌 **사업·법률·운영 판단**입니다. 각 항목에 결정해주시면 즉시 반영됩니다.

### D1. 🔴 테라피스트 합법 등재 요건 (최우선)
두바이에서 개인 출장 마사지가 합법이 되는 조건을 현지 기준으로 확정 필요.
- 옵션 A: 프리랜스 퍼밋 보유자만 등재
- 옵션 B: 라이선스 보유 업체 소속자만 등재
- 옵션 C: 자격증만 확인 (법적 리스크 큼 — 비추천)
**→ 결정: 등재 요건 서류 목록. (PRO/법무 확인 권장)**

### D2. 사업 모델 확정
리스팅 마켓플레이스: 테라피스트가 가격 설정, 플랫폼 수수료 **20%**, 파트너 법인이 결제 수취 후 80% 정산.
**→ 결정: 수수료율 20% 동의 여부, 정산 주기(주간 제안)**

### D3. 팁 정책
"기본요금 전액 업체 + 팁 의존" 구조는 불법 서비스의 경제 엔진이 되므로 배제.
**→ 결정: ① 현금 팁 금지 정책 ② 인앱 팁(상한 20~25%) 도입 여부**

### D4. 제재 정책
불법 서비스 권유 적발 시: 1회 영구퇴출 + 미정산금 몰수 (zero tolerance).
**→ 결정: 제재 기준 승인, 미스터리 쇼핑 실시 여부/주기**

### D5. 사진 승인 기준
검수 시스템은 가동 중 — 승인/반려 기준선을 운영자가 정해야 함.
- 승인: 시술 장면, 작업 공간, 자격증, 단정한 프로필
- 반려: 선정적 이미지, 시술과 무관한 전신 어필샷
**→ 결정: 기준 동의 + 검수 담당자 지정**

### D6. 환불 정책 확정
현행: 48h+ 100% / 48–24h 50% / 24h내 0%. 약관·환불 페이지·코드에 반영돼 있음.
**→ 결정: 이대로 확정 or 수정값**

### D7. Stripe 계정 (별도 요청 문서 참조)
계정 생성 + 개발자 초대만 하면 결제 개발 즉시 시작.
**→ 액션: 가입 + chohj0228@gmail.com Developer 초대**

### D8. 도메인 & 지원 이메일
현재 spa199.vercel.app + support@spa199.ae(미개통). 정식 도메인 보유/구매 여부.
**→ 결정: 도메인 + 실제 지원 이메일 주소**

### D9. 일상 운영 담당
admin 큐 4종(테라피스트 승인/사진 검수/리뷰/문의) 처리 담당자와 응답 목표시간(제안: 24h 내).
**→ 결정: 담당자 지정**
