## 참고: 결제 방식 (고객 경험)

- 고객이 예약 시 카드 정보 입력 → **즉시 청구되지 않고 승인(홀드)만** 됩니다
- 테라피스트가 수락하는 순간 실제 청구
- 거절/미응답 시 홀드 자동 해제 (고객 입장에선 청구 자체가 없음 → 분쟁/환불 민원 최소화)
- 카드 + Apple Pay + Google Pay 자동 지원, 통화는 AED
- Stripe 수수료: 결제당 2.9% + AED 1 (UAE 발급 카드 기준)

## 만약 Stripe 심사가 거절되면?

업종/법인 형태에 따라 드물게 거절될 수 있습니다. 그 경우 UAE 로컬 결제사 **Telr**로 전환합니다 (요청 서류는 동일: 트레이드 라이선스 + IBAN). 시스템은 교체 가능하게 설계되어 있어 추가 비용이 크지 않습니다.
